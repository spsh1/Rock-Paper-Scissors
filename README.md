
# Rock Paper Scissors - React Game

A simple, interactive, and responsive Rock Paper Scissors game built using React.  
This project demonstrates component-based architecture, state management using hooks, and clean UI design principles.

## How to Run Locally

Clone the repository:

```bash
git clone https://github.com/spsh1/Rock-Paper-Scissors.git

Navigate into the project folder:

cd Rock-Paper-Scissors

Install dependencies:

npm install

Start the development server:

npm start

The app will run at:
http://localhost:3000
